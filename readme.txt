=== GT White and Gold ===

== Description ==
The GT White and Gold theme is a custom theme for https://globalchange.gatech.edu.

== Installation ==
1. In Plesk, add gtwhileandgold/ to wp-content
2. In your admin panel, go to Appearance -> themes.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel
